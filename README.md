# dotfiles

Just my personal dotfiles for artix linux (runit) with suckless dwm as the window manager. It is very minimalist

## Dependencies
- `demnu` for the application launcher
- `alacritty` for the terminal emulator
- `feh` for the wallpaper
- `picom` for the compositor
- `slstatus` for the status

## Installation
To install this dotfile, you can either clone this repo and create symlinks manually.

```bash
git clone https://github.com/45M1T/dotfiles.git
